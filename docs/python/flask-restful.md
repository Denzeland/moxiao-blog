Flask-RESTful是Flask的一个扩展，它增加了对快速构建REST api的支持。它是与现有ORM/库一起工作的轻量级抽象。Flask-RESTful鼓励使用最小设置的最佳实践。如果你熟悉Flask, Flask- restful应该很容易上手

## 安装

```shell
pip install flask-restful
```
